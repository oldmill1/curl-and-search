Curl and Search

Pretty much what it says on the tin.

How to use

Say you want to know how many times the string "foo" appears in the returned HTML of www.myspace.com. Visit: http://php-curl-tool.orchestra.io/?url=www.myspace.com&search=foo

Params

url = The webpage to curl.

search = The string to search for.
