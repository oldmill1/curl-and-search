Curl and Search
===============

Pretty much what it says on the tin. 

How to use
----------

1. Go here: http://php-curl-tool.orchestra.io/
2. Then, do something like http://php-curl-tool.orchestra.io/?url=www.myspace.com&searc=foo

Params
------

**url** = The url to curl. 

**search** = The term to search for.
